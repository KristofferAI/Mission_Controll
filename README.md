# Mission Controll - Office MVP
This repository hosts the Mission Control / Office MVP with a multi-bot registry, paper betting ledger, and a Streamlit-based UI. It is flattened for a single-repo setup.

Features:
- Streamlit UI (dark theme with green accents)
- Multi-bot registry (start with a boss bot + registry)
- Bets ledger, bankroll, and jobs dashboard
- Docker optional (bootstrap.sh for local setup without Docker)

Run locally:
- Use the bootstrap.sh to install deps and start the app (no Docker)

Configuration:
- API keys and settings should be placed in a .env file. See .env.template for examples.
